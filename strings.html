🔥 JavaScript Strings (Complete Notes)

🔹 String क्या होता है?

👉 String = text (characters का group)
let name = "Nitish";

🔹 String बनाने के तरीके

let a = "Hello";   // double quotes
let b = 'Hello';   // single quotes
let c = `Hello`;   // backticks (template string)

🔥 Template Literals (IMPORTANT 🔥)

👉 backticks `` ` use होते हैं
let name = "Nitish";
let msg = `Hello ${name}`;

console.log(msg);
✅ multi-line भी लिख सकते हैं:
let text = `Line 1
Line 2`;

🔹 String Properties

🔸 length

let str = "Hello";
console.log(str.length); // 5

🔹 String Methods (IMPORTANT)

🔸 1. toUpperCase()

"hello".toUpperCase(); // HELLO

🔸 2. toLowerCase()

"HELLO".toLowerCase(); // hello

🔸 3. trim()

👉 extra space हटाता है
"  hi  ".trim(); // "hi"

🔸 4. slice()

👉 string का part निकालता है
let str = "JavaScript";
str.slice(0, 4); // Java

🔸 5. substring()

👉 similar to slice
str.substring(0, 4);

🔸 6. replace()

"Hello".replace("H", "Y"); // Yello

🔸 7. includes()

"Hello".includes("lo"); // true

🔸 8. indexOf()

"Hello".indexOf("e"); // 1

🔸 9. split()

👉 string → array
"one two three".split(" ");

🔹 String Concatenation

let a = "Hello";
let b = "World";

console.log(a + " " + b);

🔹 Escape Characters

let text = "He said \"Hello\"";
Escape Meaning
\" double quote
\' single quote
\n new line
\t tab

🔹 String is Immutable

👉 string change नहीं होती (new string बनती है)
let str = "Hello";
str[0] = "Y"; // ❌ change नहीं होगा

🔹 Access Characters

let str = "Hello";

console.log(str[0]); // H

🔹 Loop through String

for (let ch of "Hello") {
  console.log(ch);
}
