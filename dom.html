🔥🔥 HTML Collection & NodeList loops

👉 HTMLCollection = forEach नहीं चलता
👉 NodeList = सब चलता है

🔷 HTMLCollection पर कौन-कौन से loop लगते हैं?

✔️ 1. for loop (सबसे safe)

let items = document.getElementsByClassName("box");

for (let i = 0; i < items.length; i++) {
    console.log(items[i]);
}

✔️ 2. for...of loop (सबसे easy)

for (let el of items) {
    console.log(el);
}

❌ 3. forEach() (नहीं चलता)

items.forEach(el => console.log(el)); // ❌ error

🔷 3. Create / Add / Remove Elements (🔥 real DOM power)

🚀 DOM Element Create करने के Steps

🔷 ✅ Step 1: Element बनाओ

let el = document.createElement("div");
👉 अभी element memory में है (screen पर नहीं दिखेगा)

🔷 ✅ Step 2: Content डालो

el.innerText = "Hello Bhairav";
👉 text / HTML / attribute / class भी यहाँ add कर सकते हो
el.classList.add("box");

🔷 ✅ Step 3: DOM में डालो (🔥 सबसे important)

👉 यहाँ decide करते हो कहाँ दिखाना है
document.body.append(el); // last
document.body.prepend(el); // start
box.before(el); // किसी element के पहले
box.after(el); // किसी element के बाद

🧠 पूरा flow एक साथ

🔷 4. DOM Traversing (relation समझना)

🧩 Sample HTML (सब examples इसी पर based)

<div class="parent">
    <h1>Title</h1>
    <p class="para">Paragraph</p>
    <span>Span</span>
</div>

🔷 1. parentElement

let para = document.querySelector(".para");
console.log(para.parentElement);
// <div class="parent"></div>

🔷 2. children

let parent = document.querySelector(".parent");
console.log(parent.children);
// [h1, p, span] (HTMLCollection)

🔷 3. firstElementChild

console.log(parent.firstElementChild);
// </h1>

🔷 4. lastElementChild

console.log(parent.lastElementChild);
// <span>

🔷 5. nextElementSibling

let para = document.querySelector(".para");
console.log(para.nextElementSibling);
// <span>

🔷 6. previousElementSibling

console.log(para.previousElementSibling);
// <h1>

🧠 Visual समझ (🔥 बहुत important)

<div class="parent">   ← parentElement
  <h1>                ← firstElementChild
  <p>                 ← target element
  <span>              ← lastElementChild
</div>

p.previousElementSibling → h1  
p.nextElementSibling → span

🔷 5. Events (🔥🔥 सबसे important)

1️⃣ onclick (Old तरीका)

let btn = document.querySelector("button");

btn.onclick = function() {
    console.log("Clicked");
};

2️⃣ addEventListener() (🔥 Best तरीका)

element.addEventListener("eventName", function);
btn.addEventListener("click", function() {
    console.log("Clicked");
});

🔥 Multiple Event Example

btn.addEventListener("click", function() {
    console.log("First");
});

btn.addEventListener("click", function() {
    console.log("Second");
});
👉 दोनों run होंगे ✔️

🔷 Important Events

🔹 1. click

👉 जब user click करे 
let btn = document.querySelector("button");
let box = document.querySelector(".box");

btn.addEventListener("click", function() {
    box.innerText = "You clicked me!";
    box.style.backgroundColor = "yellow";
});

🔹 2. input

👉 जब user typing करे (real-time) 
let input = document.querySelector("input");
let text = document.querySelector(".text");

input.addEventListener("input", function() {
    text.innerText = input.value;
});

🔹 3. change

👉 जब value change होकर confirm हो (blur के बाद) 
let select = document.querySelector("select");
let result = document.querySelector(".result");

select.addEventListener("change", function() {
    result.innerText = "You selected: " + select.value;
});

🔹 4. submit

👉 form submit होने पर 
let form = document.querySelector("form");
let input = document.querySelector("input");

form.addEventListener("submit", function(e) {
    e.preventDefault(); // page reload रोकेगा

    if (input.value === "") {
        alert("Input cannot be empty!");
    } else {
        alert("Form submitted!");
    }
});

🔹 5. keydown

👉 keyboard key press 
let input = document.querySelector("input");

input.addEventListener("keydown", function(e) {
    if (e.key === "Enter") {
        alert("You pressed Enter!");
    }
});

🔷 6. Event Object (advance)

🔷 1. event.target (🔥 सबसे important)

👉 जिस element पर event हुआ है, उसे देता है 
let buttons = document.querySelectorAll("button");

buttons.forEach(function(btn) {
    btn.addEventListener("click", function(e) {
        e.target.style.backgroundColor = "green";
    });
});
👉 जिस button पर click करोगे, वही green होगा

🔷 2. event.type

👉 कौन सा event trigger हुआ है, बताता है 
let box = document.querySelector(".box");

function handleEvent(e) {
    if (e.type === "click") {
        box.style.backgroundColor = "yellow";
    }

    if (e.type === "mouseenter") {
        box.style.backgroundColor = "pink";
    }
}

box.addEventListener("click", handleEvent);
box.addEventListener("mouseenter", handleEvent);

🔷 3. event.preventDefault() (🔥 बहुत important)

👉 default browser behavior को रोकता है 
let form = document.querySelector("form");
let input = document.querySelector("input");

form.addEventListener("submit", function(e) {
    e.preventDefault(); // page reload रोका

    if (input.value === "") {
        alert("Fill the input first!");
    }
});

🔷 7. Event Bubbling & Capturing (🔥 concept)

1️⃣ Event Bubbling (🔥 default behavior)

<div class="parent">
  <button class="child">Click Me</button>
</div>
let parent = document.querySelector(".parent");
let child = document.querySelector(".child");

parent.addEventListener("click", function() {
    console.log("Parent clicked");
});

child.addEventListener("click", function() {
    console.log("Child clicked");
});

🧠 Output (button click करने पर):

Child clicked
Parent clicked

2️⃣ Event Capturing

parent.addEventListener("click", function() {
    console.log("Parent capturing");
}, true); // 👉 true = capturing

child.addEventListener("click", function() {
    console.log("Child clicked");
});
Parent capturing
Child clicked

3️⃣ stopPropagation() (🔥 बहुत important)

👉 event को आगे जाने से रोकता है 
child.addEventListener("click", function(e) {
    e.stopPropagation();
    console.log("Child only");
});
Child only
👉 parent तक नहीं जाएगा ❌
