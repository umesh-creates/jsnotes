🔥 JavaScript Loops

🔹 1. for loop

👉 सबसे common loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}
✅ कब use करें? जब iteration count पता हो

🔹 2. while loop

👉 पहले condition check होती है
let i = 0;

while (i < 5) {
  console.log(i);
  i++;
}
✅ कब use करें? जब condition based loop हो

🔹 3. do...while loop

👉 कम से कम 1 बार तो चलेगा ही
let i = 0;

do {
  console.log(i);
  i++;
} while (i < 5);

🔹 4. for...in loop

👉 object के keys या array index के लिए
const obj = { name: "Nitish", age: 20 };

for (let key in obj) {
  console.log(key, obj[key]);
}
⚠️ Mostly objects के लिए use करो

🔹 5. for...of loop

👉 values के लिए (array, string, iterable)
const arr = [10, 20, 30];

for (let value of arr) {
  console.log(value);
}

🔹 6. forEach() loop (array method)

👉 modern way
const arr = [1, 2, 3];

arr.forEach(function(num) {
  console.log(num);
});
👉 arrow function:
arr.forEach(num => console.log(num));

🔹 7. map() (loop + return new array)

const arr = [1, 2, 3];

const result = arr.map(n => n * 2);
console.log(result);

🔹 8. filter() (condition based loop)

const arr = [1, 2, 3, 4];

const even = arr.filter(n => n % 2 === 0);
console.log(even);

🔹 9. reduce() (advanced loop)

const arr = [1, 2, 3];

const sum = arr.reduce((acc, val) => acc + val, 0);
console.log(sum);

⚠️ Important Keywords

🔸 break

👉 loop को तुरंत रोक देता है
for (let i = 0; i < 5; i++) {
  if (i === 3) break;
  console.log(i);
}

🔸 continue

👉 current iteration skip करता है
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i);
}

🎯 Short Summary (Exam के लिए)

👉 Basic loops:
  • for
  • while
  • do...while
👉 Modern loops:
  • for...in
  • for...of
👉 Array loops:
  • forEach()
  • map()
  • filter()
  • reduce()
