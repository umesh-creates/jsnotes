🔥🔥 JS Functions

🔹 1. Function Declaration (Normal Function)

सबसे basic तरीका
function greet() {
  console.log("Hello");
}
greet();

🔹 2. Function Expression

Function को variable में store करते हैं
const greet = function() {
  console.log("Hello");
};
greet();

🔹 3. Arrow Function (=>)

Modern JS का short syntax
const greet = () => {
  console.log("Hello");
};
greet();
👉 Short form:
const add = (a, b) => a + b;

🔹 4. Anonymous Function

जिस function का कोई नाम नहीं होता
setTimeout(function() {
  console.log("Run");
}, 1000);

🔹 5. Immediately Invoked Function (IIFE)

Function जो तुरंत run हो जाता है
(function() {
  console.log("Run immediately");
})();

🔹 6. Callback Function

Function को argument के रूप में पास करते हैं
function greet(name) {
  console.log("Hello " + name);
}

function processUser(callback) {
  callback("Nitish");
}

processUser(greet);

🔹 7. Higher-Order Function

जो function दूसरे function को accept या return करे
function multiplyBy(n) {
  return function(x) {
    return x * n;
  };
}

const double = multiplyBy(2);
console.log(double(5));

🔹 8. Constructor Function

Object बनाने के लिए use होता है
function Person(name) {
  this.name = name;
}

const p1 = new Person("Nitish");
console.log(p1.name);

🔹 9. Generator Function

function* से बनता है, step-by-step values देता है
function* count() {
  yield 1;
  yield 2;
  yield 3;
}

const c = count();
console.log(c.next().value);

🔹 10. Async Function

Asynchronous code handle करने के लिए
async function fetchData() {
  return "Data";
}

🔹 11. Recursive Function

जो खुद को call करता है
function factorial(n) {
  if (n === 1) return 1;
  return n * factorial(n - 1);
}
