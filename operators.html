🔥 JavaScript Operators (Complete Notes)

🔹 Operator क्या होता है?

👉 Operator एक symbol होता है जो values पर operation करता है
let sum = 5 + 3;

🔹 1. Arithmetic Operators (गणित वाले)

Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus (remainder)
** Power
++ Increment
-- Decrement 
let a = 10, b = 3;

console.log(a + b); // 13
console.log(a % b); // 1
console.log(a ** 2); // 100

🔹 2. Assignment Operators

Operator Example
= a = 5
+= a += 2
-= a -= 2
*= a *= 2
/= a /= 2
%= a %= 2 
let x = 5;
x += 3; // 8

🔹 3. Comparison Operators

👉 result हमेशा true/false होता है
Operator Meaning
== equal (loose)
=== equal (strict)
!= not equal
!== strict not equal
> greater
< less
>= greater equal
<= less equal 
console.log(5 == "5");  // true
console.log(5 === "5"); // false

🔥 ⚠️ IMPORTANT (Interview)

👉 == vs ===
  • == → type check नहीं करता
  • === → type भी check करता
✅ Always use ===

🔹 4. Logical Operators

Operator Meaning
&& AND
|| OR
! NOT 
let age = 20;

console.log(age > 18 && age < 25); // true

🔹 5. Bitwise Operators (Advanced)

Operator Meaning
& AND (bitwise)
|| OR (logical)
^ XOR (bitwise)
~ NOT (bitwise)
<< Left shift
>> Right shift 
👉 beginners के लिए optional

🔹 6. Ternary Operator

👉 short if-else
let age = 18;

let result = age >= 18 ? "Adult" : "Minor";

🔹 7. Type Operators

Operator Meaning
typeof type check
instanceof object type check 
console.log(typeof "Hello"); // string

🔹 8. String Operator

let first = "Hello";
let second = "World";

console.log(first + " " + second);

🔹 9. Nullish & Optional (Modern 🔥)

🔸 Nullish Coalescing ??

let name = null;

console.log(name ?? "Guest");
👉 null/undefined पर fallback देता है

🔸 Optional Chaining ?.

let user = {};

console.log(user?.name);
👉 error नहीं देगा
