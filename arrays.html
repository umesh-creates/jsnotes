🔥 JavaScript Arrays (Complete Notes)

🔹 Array क्या होता है?

👉 Array = multiple values को एक variable में store करना
let arr = [1, 2, 3, 4];

🔹 Array Create करने के तरीके

let a = [1, 2, 3];
let b = new Array(1, 2, 3);

🔹 Index (IMPORTANT)

👉 Array index 0 से शुरू होता है
let arr = ["a", "b", "c"];

console.log(arr[0]); // a

🔹 Array Properties

🔸 length

let arr = [1, 2, 3];
console.log(arr.length); // 3

🔥 Important Array Methods

🔹 1. push() (end में add)

let arr = [1, 2];
arr.push(3); // [1,2,3]

🔹 2. pop() (end से remove)

arr.pop();

🔹 3. unshift() (start में add)

arr.unshift(0);

🔹 4. shift() (start से remove)

arr.shift();

🔹 5. slice() (copy part)

let arr = [1, 2, 3, 4];
arr.slice(1, 3); // [2,3]

🔹 6. splice() (add/remove/change 🔥)

let arr = [1, 2, 3];

arr.splice(1, 1); // remove
arr.splice(1, 0, 5); // add

🔹 7. concat() (join arrays)

let a = [1];
let b = [2];

a.concat(b); // [1,2]

🔹 8. includes()

[1,2,3].includes(2); // true

🔹 9. indexOf()

[1,2,3].indexOf(2); // 1

🔥 Looping Arrays (VERY IMPORTANT)

🔹 for loop

let arr = [1,2,3];

for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}

🔹 for...of

for (let val of arr) {
  console.log(val);
}

🔹 forEach()

arr.forEach(val => console.log(val));

🔥 Advanced Methods (IMPORTANT 🔥)

🔹 map()

👉 new array return करता है
let arr = [1,2,3];

let result = arr.map(n => n * 2);

🔹 filter()

let even = arr.filter(n => n % 2 === 0);

🔹 reduce()

let sum = arr.reduce((acc, val) => acc + val, 0);

🔹 Multidimensional Array

let arr = [
  [1,2],
  [3,4]
];

console.log(arr[0][1]); // 2

🔹 Array Destructuring (Modern 🔥)

let arr = [10, 20];

let [a, b] = arr;

🔹 Spread Operator ...

let arr1 = [1,2];
let arr2 = [...arr1, 3, 4];

🔹 Array is Mutable

👉 Array change हो सकता है
let arr = [1,2];
arr[0] = 10;
