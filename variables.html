🔥 JavaScript Variables (Complete Notes)

🔹 Variable क्या होता है?

👉 Variable एक container होता है जिसमें data store किया जाता है
let name = "Nitish";

🔹 Variable Declare करने के तरीके

✅ 1. var (Old तरीका)

var x = 10;

🔸 Features:

  • function scoped
  • re-declare ✅
  • update ✅
⚠️ Problem: unpredictable behavior

✅ 2. let (Modern)

let age = 20;

🔸 Features:

  • block scoped ✅
  • re-declare ❌
  • update ✅

✅ 3. const (Constant)

const pi = 3.14;

🔸 Features:

  • block scoped ✅
  • re-declare ❌
  • update ❌
⚠️ Value change नहीं कर सकते

🔥 Difference (VERY IMPORTANT)

Feature var let const
Scope function block block
Re-declare ✅ ❌ ❌
Update ✅ ✅ ❌

🔹 Scope क्या होता है?

🔸 1. Global Scope

let a = 10;

🔸 2. Block Scope

{
  let x = 5;
}

🔸 3. Function Scope

function test() {
  var y = 20;
}

🔹 Hoisting (IMPORTANT)

👉 JavaScript variables को ऊपर उठा देता है
console.log(a); // undefined
var a = 10;
👉 let और const में error आता है
console.log(a); // ❌ error
let a = 10;

🔹 Naming Rules

✅ सही:
let userName;
let _age;
let $price;
❌ गलत:
let 123name; ❌
let var; ❌

🔹 Data Types (Short)

let name = "Nitish";   // string
let age = 20;         // number
let isTrue = true;    // boolean
let x;                // undefined
let y = null;         // null

🔹 Dynamic Typing

👉 JS में type change हो सकता है
let a = 10;
a = "Hello";

🔹 Best Practices

✅ const use करो (default) ✅ let use करो
(अगर value change करनी हो) ❌ var avoid करो
