🔥 JavaScript Objects (Complete Notes)

🔹 Object क्या होता है?

👉 Object = key-value pair का collection
let person = {
  name: "Nitish",
  age: 20
};

🔹 Object Create करने के तरीके

✅ 1. Object Literal (most common)

let obj = {
  name: "Rahul",
  age: 25
};

✅ 2. Constructor से

let obj = new Object();
obj.name = "Rahul";

✅ 3. Constructor Function

function Person(name, age) {
  this.name = name;
  this.age = age;
}

let p1 = new Person("Nitish", 20);

🔹 Object Access कैसे करें

🔸 Dot notation

console.log(person.name);

🔸 Bracket notation

console.log(person["age"]);

🔹 Add / Update / Delete

let obj = { name: "Nitish" };

// add
obj.age = 20;

// update
obj.name = "Rahul";

// delete
delete obj.age;

🔹 Object Methods (function inside object)

let user = {
  name: "Nitish",
  greet: function() {
    console.log("Hello");
  }
};

user.greet();

🔥 this keyword (IMPORTANT 🔥)

👉 current object को refer करता है
let user = {
  name: "Nitish",
  greet() {
    console.log(this.name);
  }
};

🔹 Loop in Object

🔸 for...in

let obj = { a:1, b:2 };

for (let key in obj) {
  console.log(key, obj[key]);
}

🔹 Object Methods (built-in)

🔸 Object.keys()

Object.keys(obj); // ["a","b"]

🔸 Object.values()

Object.values(obj); // [1,2]

🔸 Object.entries()

Object.entries(obj);

🔹 Nested Object

let user = {
  name: "Nitish",
  address: {
    city: "Nagpur"
  }
};

console.log(user.address.city);

🔹 Optional Chaining ?.

console.log(user?.address?.city);

🔹 Destructuring (Modern 🔥)

let user = { name: "Nitish", age: 20 };

let { name, age } = user;

🔹 Spread Operator

let obj1 = { a:1 };
let obj2 = { ...obj1, b:2 };

🔹 Object is Mutable

let obj = { a:1 };
obj.a = 10;

🔹 JSON (IMPORTANT 🔥)

👉 Object का string format
let user = { name: "Nitish" };

let json = JSON.stringify(user);
let obj = JSON.parse(json);
