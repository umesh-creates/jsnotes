🔥🔥 fetch() - insert, update, delete

Syntax :

fetch(file/URL, { 
    method: 'post', //insert - post, update - put, delete - DELETE & read - GET (default)
    body: Data, // server पर जो data method में से कोई करना है (form data/json data/text)
    header : {
        'Content-Type' : 'application/json', // json data
        // or 
        'Content Type' : 'application/x-www-form-urlencoded' // form data 
        // server पर जो data भेजा जा रहा है उसका type क्या है 
    }
})

Example : 

let obj = {
    name: 'kiran',
    age: '87',
    city: 'delhi',
  }
fetch('https://jsonplaceholder.typicode.com/posts', {
  method: 'POST',
  body: JSON.stringify(obj), // 🔄 JavaScript object को string (JSON format) में बदल देता है
  headers: {
    'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));
OUTPUT { "name": "kiran", "age": "87", "city": "delhi", "id": 101 }

🧠 event.preventDefault() क्या है?

👉 यह एक method है जो:

❌ browser का default behaviour रोक देता है

🤔 Default behaviour क्या होता है?

👉 Browser कुछ काम अपने आप करता है

Examples:

Action Default Behaviour
Form submit Page reload 🔄
Link (<a>) click New page open 🌐
Button submit Form submit

Alternative - special for HTML Form 

document.getElementById('save').addEventListener('click', (event) => {
    event.preventDefault();

    let name = document.getElementById('name').value;
    let age = document.getElementById('age').value;
    let city = document.getElementById('city').value;

    fetch('https://jsonplaceholder.typicode.com/posts', {
        method: 'POST',
        body: JSON.stringify({
            name: name,
            age: age,
            city: city
        }),
        headers: {
            'Content-type': 'application/json',
        },
    })
    .then(response => response.json())
    .then(json => console.log(json));
});
